# Simbas
Simbas merupakan aplikasi desktop berbasis Java Swing yang dirancang untuk membantu pemilik toko sembako di Kota Malang dalam mengelola data stok barang dan transaksi penjualan secara digital dan terkomputerisasi. Aplikasi ini menggantikan sistem pencatatan manual yang selama ini sering mengakibatkan kesalahan tulis dan kesalahan hitung.
